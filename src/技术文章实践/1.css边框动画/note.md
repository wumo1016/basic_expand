## border

- border-width
  - 1.明确指定宽度值 如：px|em|rem 等
  - 2.使用全局关键字
    - thin: 细边线
    - medium: 中等边线
    - thick: 宽边线
- border-style
  - none/hidden: 不显示 边框为 0
  - solid: 实线
  - dashed: 线段虚线
  - dotted: 圆点虚线
  - double: 双实线
  - inset: 陷入效果
  - outset: 突出效果

## :root

- 代表根元素

## css 变量

- 声明 css 变量 必须以`--`开头
- 声明的变量只能在作用域 dom 作用域内使用
