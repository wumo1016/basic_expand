## API

- $.fn
  - 在`$.fn`上定义的属性方法等会挂在到 jquery 的原型上, 所有 jquery 的实例都可以获取
- [$.extend](https://www.jquery123.com/jQuery.extend/)
  - 作用: 将两个或更多对象的内容合并到第一个对象
    - 第一个如果 true, 则代表深度合并
    - 如果只有一个参数, 则默认目标对象为 jquery 自身
- 实例的 sortable 方法来自于 jquery UI
- show()/hide()
  - 设置元素的显示与隐藏
- empty()
  - 清空 dom
