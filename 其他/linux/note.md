## 常用命令

- `ls -a`: 显示所有文件, 包含隐藏文件
- `mkdir [文件夹名]`: 创建文件夹
- `touch [文件名]`: 创建文件
- `rm -rf [文件夹名]`: 删除文件夹, 包含子文件夹
- `rm -f [文件名]`: 删除文件

## vim 命令

- 启动与退出
  - `vim [文件名]`: 打开文件
  - `:q`: 退出
  - `:q!`: 强制退出
  - `:w`: 保存文件
  - `:wq`: 保存并退出 vim
  - `:wq!`: 强制保存并退出 vim
  - `:%d`: 清空内容
- 光标移动
  - `a`: 在光标前插入
  - `i`: 在光标后插入
  - `I`: 在行首插入
  - `A`: 在行尾插入
  - `o`: 在下方新开一行插入
  - `O`: 在上方新开一行插入
  - `Esc`: 退出插入模式
- 光标移动
  - `kjhl`: 分别对应上、下、左、右移动
  - `0`: 移动到行首
  - `$`: 移动到行尾
  - `gg`: 移动到文件开头
  - `G`: 移动到文件结尾
  - `nG`: 移动到第 n 行
  - `Ctrl + f`: 向下滚动一页
  - `Ctrl + b`: 向上滚动一页
  - `Ctrl + u`: 向上滚动半页
  - `Ctrl + d`: 向下滚动半页
- 编辑操作
  - `x`: 删除当前字符
  - `dd`: 删除当前行
  - `d$`: 删除从光标到行尾的内容
  - `d0`: 删除从光标到行首的内容
  - `yy`: 复制当前行
  - `y$`: 复制从光标到行尾的内容
  - `y0`: 复制从光标到行首的内容
  - `p`: 粘贴(在光标下一行)
  - `P`: 粘贴(在光标上一行)
  - `u`: 撤销
  - `Ctrl + r`: 重做
